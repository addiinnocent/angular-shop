# Webshop MEAN stack

Ein Projekt von der Coding Stübchen Community.
Bevor das Programm mit dem Command 'npm start' gestartet werden kann, müssen folgende Programme installiert werden (npm install).

# Dependencies:

Programme:
- Angular CLI
- NodeJS
- npm
- MongoDB

Module (included):
- express
- mongoose
